# Random Helpful Resources

## Statistics and data analysis
  - Applied Multivariate Statistics in R (ebook): https://uw.pressbooks.pub/appliedmultivariatestatistics/
  - Numerical Ecology with R (ebook, available with UIC Library account): https://link.springer.com/book/10.1007/978-3-319-71404-2

## Data visualization resources
  - R Chart Colors: https://r-charts.com/colors/ 
    - Lists all color names that R recognizes
    - Also lists many predefined palettes for multi-colored plots
  - CV Simulator: https://asada.website/cvsimulator/e/
    - App for Andoid and iOS that simulates different types of color blindess in real time. You can look at your computer screen through your phone camera to see if the color combo you're using is accessible to color-blind viewers.
